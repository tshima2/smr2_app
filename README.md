# README

# アプリ名 #
SmoothRuninng-v2 (現場情報共有ツール v2)  

# 概要 #
記録したい場所を地図ベースで登録し、気づきなどを蓄積・共有

# コンセプトa#
未経験・経験浅の物流業従事者を応援！
現場に対する気づきを蓄積し、自らの成長や所属チーム内での活用を図ります。

# バージョン #
Ruby 2.6.3 Rails 6.0.4

# 機能一覧 #
	* 地図上で位置を指定することで現場情報を作成	
	* 地図上から現場情報の参照、編集、削除
	* 現場情報をチーム内で共有し、コメントや画像の追加
	* 現場情報にラベルやハッシュタグを付加し、絞り込み表示
